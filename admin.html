Music rotation

While ANY custom tracks exist here, players hear ONLY these (rotating). Delete them all and the game reverts to the built-in synthesized tracks. The name you enter is what players see in their "Music track" dropdown.

Special cities per map

Maximum number of special cities (Great Beacon, Iron Mine, Twin Gates…) that can appear on a map. 0 disables them; small maps may spawn fewer than the maximum.

🧪 Test Lab — jump into any battle

Opens the game in a new tab in test mode: pick any level and settings and play it immediately. A throwaway "⚙ Test Pilot" profile is used — nothing is saved (no merit, XP, unlocks, or leaderboard changes), so you can test freely without polluting real player data.

The game file must sit in the same folder as this console. Ad/music/difficulty settings above only reach the test tab after 💾 Save (the tier, speed, and special-city overrides here are passed directly and work regardless).

Players

Local mode lists players saved in this browser; server mode lists all cloud players.

Local data tools

Exports every player profile and setting stored in this browser to a JSON file (your "user state file"). Import restores it — e.g., to move data to another machine or keep backups.
